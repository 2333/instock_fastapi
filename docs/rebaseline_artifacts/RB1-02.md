- 已核对 `api/index.ts`、`api/alert.ts` 与 `useAnalytics.ts`

## 关键结论

### `keep`

- Routes / views:
  - `Login.vue`
  - `Dashboard.vue`（页面保留，内部需裁剪）
  - `Workspace.vue`
  - `Stocks.vue`
  - `StockDetail.vue`（页面保留，埋点依赖需裁剪）
  - `Patterns.vue`
  - `Backtest.vue`（页面保留，内部需裁剪）
  - `Selection.vue`
  - `Attention.vue`（页面保留，内部需裁剪）
  - `Settings.vue`
- Composables / shared:
  - `useAuth.ts`
  - `useLocale.ts`
  - `useResizablePanel.ts`
  - `useUserPreferences.ts`
  - `utils/patternLabels.ts`
  - `components/charts/*`
- API surface:
  - `authApi`
  - `attentionApi`
  - `backtestApi`
  - `marketApi`
  - `selectionApi`
  - 其余 `M0` 主链路必需接口

理由:
- 上述页面和基础能力构成 `M0` 的用户主链路。
- `Dashboard.vue`、`Backtest.vue`、`Attention.vue` 属于 mixed view，页面本体保留，但超范围分支需要剥离。

### `remove`

- Route / views:
  - `Alerts.vue`
  - `Optimization.vue`
  - `Reports.vue`
  - `ReportDetail.vue`
  - `ReportPreferences.vue`
  - `StrategyMarket.vue`
- Components:
  - `NotificationBell.vue`
  - `StrategyCard.vue`
  - `StrategyDetailModal.vue`
  - `StrategyLeaderboard.vue`
- Composables:
  - `useAnalytics.ts`
- API:
  - `api/alert.ts`
  - `alertApi`
  - `strategySocialApi`
  - `optimizationApi`
  - `reportApi`

理由:
- 这些页面、组件、接口直接承接 `P3-01/03/04/05/06` 的运行时能力，不属于 `M0`。
- `useAnalytics.ts` 属于 `P3-01` 埋点底座，应与 `events_router` 一并撤回。

### `hold`

- Mixed routes / views / components:
  - `router/index.ts`
  - `Dashboard.vue`
  - `Backtest.vue`
  - `Attention.vue`
  - `StockDetail.vue`
  - `Selection.vue`
  - `components/layout/AppHeader.vue`
  - `api/index.ts`

理由:
- `router/index.ts` 同时包含 `M0` 路由与超范围路由，需先局部删入口。
- `Dashboard.vue` 同时展示市场概览和 `reports/optimization/strategySocial` 卡片，必须裁剪而不是删除。
- `Backtest.vue` 同时包含 `my/public/optimization` 三类标签源，需保留基础回测流，删除公共策略和参数优化分支。
- `Attention.vue` 页面本体属于 `M0`，但快速预警弹窗与 `alertApi` 调用应剥离。
- `AppHeader.vue` 仅因 `NotificationBell` 混入超范围能力，应局部清理。
- `api/index.ts` 需要删掉超范围导出，但文件本身保留。

### 重点文件判定

- `Dashboard.vue`: `hold -> keep(after trim)`
  - 保留市场温度计、我的关注、今日扫描、基础回测卡片
  - 移除报告、优化、公共策略相关数据加载与跳转
- `Backtest.vue`: `hold -> keep(after trim)`
  - 保留 `my` 模板、运行回测、历史对比、URL 状态
  - 移除 `public`、`optimization` 标签页及相关 API
- `Alerts.vue`: `remove`
- `Optimization.vue`: `remove`
- `Reports.vue` / `ReportDetail.vue` / `ReportPreferences.vue`: `remove`
- `StrategyMarket.vue`: `remove`

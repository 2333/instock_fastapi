# InStock 智能股票分析平台 - 系统架构文档

## 1. 系统概述

InStock是一个基于FastAPI和Vue3构建的智能股票分析平台，提供股票数据采集、技术分析、策略选股和回测等功能。

## 4. 定时任务流程

```
┌──────────────────────────────────────────────────────────────────────────┐
│                           定时任务执行流程                                │
├──────────────────────────────────────────────────────────────────────────┤
│                                                                           │
│   时间        任务                    依赖数据           输出             │
│   ────        ────                    ────────           ────             │
│                                                                           │
│   15:30    fetch_daily_data     →   东方财富API    →   stocks, daily_bars │
│            (数据抓取)                   │                                   │
│                                        ↓                                   │
│   16:00    fetch_fund_flow      →   stocks        →   fund_flows,        │
│            (资金流向)                   │              sector_fund_flows  │
│                                        ↓                                   │
│   17:00    calculate_indicators →   daily_bars    →   indicators         │
│            (指标计算)                   │                                   │
│                                        ↓                                   │
│   17:30    run_pattern_         →   daily_bars,   →   patterns          │
│            recognition             indicators                           │
│            (形态识别)                   │                                   │
│                                        ↓                                   │
│   18:00    run_strategy_        →   patterns,     →   strategy_results  │
│            selection               indicators                           │
│            (策略选股)                   │                                   │
│                                        ↓                                   │
│   周日03:00 cleanup_old_data    →   所有时序表    →   清理过期数据        │
│            (数据清理)                                                       │
│                                                                           │
└──────────────────────────────────────────────────────────────────────────┘
```

## 5. 技术栈

### 5.1 后端技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Python | 3.11+ | 主要开发语言 |
| FastAPI | 0.104+ | Web框架 |
| SQLAlchemy | 2.0+ | ORM |
| AsyncPG | 0.29+ | 异步PostgreSQL驱动 |
| PostgreSQL | 15+ | 主数据库 |
| TimescaleDB | 2.x | 时序数据扩展 |
| Redis | 7.x | 缓存/任务队列 |
| APScheduler | 3.x | 定时任务调度 |
| TA-Lib | 0.4+ | 技术指标计算 |
| Pandas | 2.x | 数据处理 |
| NumPy | 1.x | 数值计算 |

### 5.2 前端技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Vue.js | 3.4+ | 前端框架 |
| TypeScript | 5.x | 类型安全 |
| Vite | 5.x | 构建工具 |
| Pinia | 2.x | 状态管理 |
| Vue Router | 4.x | 路由管理 |
| Axios | 1.x | HTTP客户端 |
| ECharts | 5.x | 图表可视化 |
| Element Plus | 2.x | UI组件库 |

### 5.3 基础设施

| 技术 | 用途 |
|------|------|
| Docker | 容器化部署 |
| Docker Compose | 多容器编排 |
| Nginx | 反向代理/静态资源 |
